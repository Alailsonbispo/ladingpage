# RITMO E CÓDIGO // A Primeira Track

STATUS: Deploy em Produção Gênero: Landing Page Pessoal / CV Digital

Este projeto é a base de tudo, minha Primeira Track. Foi aqui que o estilo e a filosofia do THE VAULT foram definidos: código limpo, visual de alto contraste e foco total na tipografia.

O objetivo não era apenas construir uma página, mas estabelecer a minha identidade visual como desenvolvedor: organizada, direta e com estilo próprio.

## O Que Você Encontra Aqui (Features)

HTML Semântico: Estrutura clara e organizada para acessibilidade e SEO.

Estilo P&B: Uso estratégico de cores de alto contraste para gerar impacto imediato.

Tipografia Assertiva: Escolha de fontes que traduzem a vibe Streetwear e P&B.

Responsividade: Garantia de que o Ritmo e Código carrega perfeitamente em qualquer dispositivo.
